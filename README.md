 Personal Portfolio

  A fully responsive personal portfolio website built using **HTML** and **CSS**.  
 The website showcases my **projects, skills, education, and contact information** in a clean and structured layout.

 Features
- Responsive design for all devices  
- Clean, modern layout  
- Sections: About, Skills, Projects, Contact  
- Fast and lightweight (no frameworks used)  
- Assets optimized for web  

 Tech Stack
- HTML5  
- CSS3  
- Responsive design (Flexbox, Grid)  
- Fonts & Icons
